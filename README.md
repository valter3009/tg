# GidMeteo - Weather WebApp

Веб-приложение для отображения погоды, созданное для Telegram WebApp.

## Описание

Это веб-приложение предоставляет пользователям удобный интерфейс для просмотра информации о погоде в Telegram.

## Файлы

- **index.html** - Основное веб-приложение с погодным виджетом
- **meteo.html** - Альтернативная версия веб-приложения

## Особенности

- Интеграция с Telegram WebApp API
- Адаптивный дизайн для мобильных устройств
- Яндекс.Метрика для отслеживания статистики
- Полностью клиентское приложение (только HTML/CSS/JavaScript)

## Использование

Просто откройте файлы `index.html` или `meteo.html` в браузере или интегрируйте их в Telegram бота как WebApp.

## Деплой

Вы можете разместить эти файлы на любом хостинге статических файлов:
- GitHub Pages
- Netlify
- Vercel
- или любой другой веб-сервер
